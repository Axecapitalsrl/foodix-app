# Foodix

Landing / web de la app de delivery **Foodix** — pedí lo que se te antoje, llega volando.

Nacimos en **Santa Cruz de la Sierra, Bolivia**, con proyección de expansión a La Paz, Cochabamba y el resto de Latinoamérica.

## Estructura

Sitio estático — se despliega tal cual en Vercel (sin build).

```
index.html            # Home
assets/
  foodix-logo.png       # Logo (fondo transparente, recortado)
  foodix-logo-fondo.png # Logo con fondo oscuro
```

## Desarrollo local

```bash
python -m http.server 8000
# abrir http://localhost:8000
```
